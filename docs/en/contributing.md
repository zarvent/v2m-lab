# ❤️ CONTRIBUTING

Thanks for your interest in contributing to this project!

All help is welcome, whether it's reporting bugs, suggesting improvements, or adding new features.

### HOW TO CONTRIBUTE

1.  **Open an issue** to discuss the change you want to make
2.  **Fork** the repository and create a new branch for your changes
3.  **Make sure** your code follows the project's style and conventions
4.  **Submit a pull request** with a clear description of your changes
